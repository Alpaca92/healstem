## 바코드 입력 엑셀 물품명

1. 물품명의 구분은 반드시 `,`으로 한다

```
ex. -석미화 싱3,싱카3,의2,의카2,슈3(개인물건 슈카3)
```

2. 행사 등으로 인한 수량은 괄호안에 세부내용을 작성한다

```
⭕ ex. 메20(18+2)
❌ ex. 메18+2
```

3. 줄임말은 반드시 아래의 표와 같이 작성한다
